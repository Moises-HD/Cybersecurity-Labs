# 🚀 Secure Deployment & DevSecOps — Cybersecurity Master's Program

This module focuses on the secure deployment of applications and infrastructure, integrating DevOps practices with security controls (DevSecOps).

It covers CI/CD pipelines, containerization, infrastructure automation, and secure software development practices.

---

## 🧠 Lab Overview

These labs demonstrate how to securely build, test, deploy, and monitor applications in production environments, combining development workflows with security validation.

---

## 📂 Lab Summary

### 🧪 Secure Development & Unit Testing (PHP)

- Installation and configuration of PHP, PHPUnit, and development environment (VS Code)
- Implementation of unit testing for custom functions (e.g., Fibonacci)
- Test automation and validation using assertions

**Key concepts:**
- Test-driven development (TDD)
- Code structure and maintainability
- Automated validation

---

### 🔓 Web Application Security Testing (DVWA)

- Deployment and configuration of DVWA environment
- Analysis of SQL Injection behavior under malformed inputs
- Comparison between insecure (Low) and secure (Impossible) implementations

**Key concepts:**
- Input validation
- SQL Injection exploitation
- Secure coding practices

---

### 🌐 Web Vulnerabilities (XSS & SQL Injection)

- Stored XSS attacks:
  - Message injection
  - Redirection attacks
  - Session cookie theft

- Analysis of different attack types:
  - Stored / Reflected XSS
  - Blind SQL Injection

**Key concepts:**
- OWASP Top 10
- Client-side attacks
- Session hijacking

---

### 📱 Mobile Security Analysis

- Comparison of mobile permission models
- Secure data storage techniques
- Detection of insecure network protocols
- Binary inspection for sensitive data leaks
- CASB implementation scenarios (BYOD environments)

**Key concepts:**
- Mobile application security
- Data protection
- Network traffic analysis

---

### ⚙️ DevOps & Secure Deployment

- Version control systems and access management
- CI/CD pipeline setup and integration
- Docker containerization and testing
- Automated deployment workflows
- System resilience and failure handling
- Disaster recovery planning

**Key concepts:**
- DevOps & DevSecOps
- Infrastructure as Code (IaC)
- Automation and scalability
